# Gety Skills

Agent skills for [Gety](https://gety.ai) — connect AI agents to your local documents.

<p align="center">
  <img src="https://img.shields.io/badge/Skills-1-blue" alt="1 Skills" />
</p>

## Available Skills

| Skill | Path | Description |
|-------|------|-------------|
| gety-cli | `skills/gety-cli` | Search and retrieve local documents via Gety CLI |

## Installation

Install a specific skill:

```bash
npx skills add gety-ai/gety-skills --path skills/gety-cli
```

## Requirements

- [Gety](https://gety.ai) desktop app installed and running
- Enable AI integrations in [Gety](https://gety.ai) desktop app settings.
